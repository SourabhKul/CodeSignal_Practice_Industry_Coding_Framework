# Live Coding Drills

Standalone 45-60 minute exercises for a CodeSignal-style live coding session. These are not progressive assessments: read the prompt, implement the candidate `solution.py`, and run its tests directly.

| Drill | Focus |
| --- | --- |
| `web_crawler` | BFS, URL filtering, concurrency |
| `lru_cache` | eviction, persistence, thread safety |
| `duplicate_file_finder` | recursive I/O, grouping, chunked hashing |
| `profiler_denoising` | interval reconstruction from stack samples |
| `tokenizer_review` | debugging, edge cases, deterministic token matching |

For example:

```bash
cd live_rounds/web_crawler
python3 test_solution.py
```
