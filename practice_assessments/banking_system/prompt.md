# Banking System

This drill follows the CodeSignal Industry Coding Framework style:

- one project-based task
- four progressive levels
- method contracts from earlier levels remain valid
- later levels add extension methods and require refactoring

Read the level files in order:

1. `level1.md`
2. `level2.md`
3. `level3.md`
4. `level4.md`

Implement only `solution.py`.

Run one level at a time:

```bash
python3 -m unittest test_solution.TestBankingSystem.test_level_1_accounts
python3 -m unittest test_solution.TestBankingSystem.test_level_2_top_spenders
python3 -m unittest test_solution.TestBankingSystem.test_level_3_transfers
python3 -m unittest test_solution.TestBankingSystem.test_level_4_merge_history
```
