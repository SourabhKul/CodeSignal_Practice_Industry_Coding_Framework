# Parcel Manager

This is an original progressive practice exercise based on a reported parcel-manager assessment shape. No credible public copy of the exact prompt was found, so the contracts here are a testable reconstruction rather than proprietary interview content.

Read `level1.md` through `level4.md` one at a time. Implement `solution.py` and run only the current level test:

```bash
python3 -m unittest test_solution.TestParcelManager.test_level_1_inventory
python3 -m unittest test_solution.TestParcelManager.test_level_2_inventory_queries
python3 -m unittest test_solution.TestParcelManager.test_level_3_courier_gated_status
python3 -m unittest test_solution.TestParcelManager.test_level_4_version_rollback
```
