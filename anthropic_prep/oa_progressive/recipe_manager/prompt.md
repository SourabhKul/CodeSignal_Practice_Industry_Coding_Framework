# Recipe Manager

This is an original practice variant based on the publicly reported Anthropic recipe-manager family, not a verbatim assessment prompt.

Read `level1.md` through `level4.md` one at a time. Implement `solution.py` and run only the current level test:

```bash
python3 -m unittest test_solution.TestRecipeManager.test_level_1_recipes
python3 -m unittest test_solution.TestRecipeManager.test_level_2_search
python3 -m unittest test_solution.TestRecipeManager.test_level_3_users
python3 -m unittest test_solution.TestRecipeManager.test_level_4_history
```
