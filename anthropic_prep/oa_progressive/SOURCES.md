# Sources and Evidence

Reviewed July 9, 2026. Official sources establish format; candidate reports only support scenario likelihood.

## Official CodeSignal Format

- [Industry Coding Assessment rules](https://support.codesignal.com/hc/en-us/articles/19116922232983-What-are-the-Industry-Coding-Assessment-ICA-rules)
  - One domain-agnostic project, four progressive levels, 90 minutes, backward-compatible refactoring.
- [Creating custom test cases](https://support.codesignal.com/hc/en-us/articles/8105682814487-Creating-custom-test-cases-for-single-function-questions)
  - Standard and hidden tests determine scoring and level progression.
- [Supported ICA environments](https://support.codesignal.com/hc/en-us/articles/10656138860823-What-languages-environments-are-available-per-certified-assessment)
  - Lists Python 3 without promising a fixed patch version.
- Local `CodeSignal Skills Evaluation Framework.pdf`
  - Detailed Level 1-4 design guidance and file-storage example.

## Interpretation Rule

A reported scenario is not an official guarantee. Similar CodeSignal questions are reused across companies, exact APIs vary, and some reports combine OAs with later live interviews. This pack labels uncertainty instead of claiming complete knowledge of a private rotating bank.
