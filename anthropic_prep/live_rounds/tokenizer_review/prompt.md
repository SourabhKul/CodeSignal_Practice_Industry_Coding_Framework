# Live Drill: Tokenizer Code Review

Fix `Tokenizer` in `solution.py`.

The implementation is intentionally flawed. Identify and fix issues such as:

- Literal unknown-token text appearing in normal input.
- Overlapping vocabulary tokens.
- Round-tripping text through `tokenize` and `detokenize`.
- Empty input.

The goal is partly code review: explain the bugs before patching them.
